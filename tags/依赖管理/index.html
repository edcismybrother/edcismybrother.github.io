dep依赖管理

Sep 28, 2018 by 皮智文
Tags: 依赖管理
安装dep go get -u github.com/golang/dep/cmd/dep dep基本使用 dep init 执行这个命令会将当前目录下所有文件import的第三方工具进行打包 这个建议加上-v ,可以看日志 init是从网络上去下载对应的中间件，包很多的话，下载会非常慢 dep init -gopath 会先去本地找对应的包，没有的话，再去网络上下载 dep ensure -update 依据Gopkg.toml文件的信息获取制定版本的依赖 Gopkg.toml配置小结: [[constraint]] name = "github.com/panjf2000/ants" version = "=3.6.0" 上述引用为例子: "3.6.0" 约束使用 3.6.0 - 3.7.0 之间的最新版 "<=3.6.0" 约束使用最高版本为 3.6.0 "=3.6.0" 约束指定为3.6.0
